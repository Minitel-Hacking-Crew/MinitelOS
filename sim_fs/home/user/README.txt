Aide memoire utilisateur :

Pour personnaliser votre message du jour :
  1. Creez le fichier motd_perso.txt dans votre repertoire
  2. Ajoutez vos lignes de texte
  3. Les commandes shell (echo, date, uptime...) sont
     executees automatiquement pour l'affichage dynamique
  4. Tapez 'motd' pour voir le resultat
